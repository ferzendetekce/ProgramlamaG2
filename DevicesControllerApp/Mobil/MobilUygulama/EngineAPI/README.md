Bu repoda projenin tüm bileşenlerini içeren bir şablon oluşturulmuştur. Tüm gruplar bu projeyi pull işlemiyle yerel ortamlarına indirip kendi form veya sınıflarını (class) güncelleyebilirler. Güncellemelerin ardından değişiklikleri push ederek geri yükleyebilirler. Yaptığınız geliştirmelere ait kodları mutlaka kendi bilgisayarınızda ayrıca yedekleyiniz. Size atanan formları geliştirirken ihtiyaç duyduğunuz fonksiyonları DatabaseManager.cs dosyasına ekleyiniz. Ayrıca Issues bölümünü kullanarak grup arkadaşlarınız ve diğer gruplarla tartışabilir, soru sorabilir ve taleplerde bulunabilirsiniz.
